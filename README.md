## Build Setup:
```bash
# Download repository:

$ git clone https://github.com/WhiteDevilMan/FSD

# Go to the folder:
$ cd FSD

# Install dependencies from package.json:
$ npm install

or

$ npm i

# Start server with hot reload
$ npm run dev

# Build project. Output will be at 'dist' folder
$ npm run build

# Deploy project to gh-pages (after "npm run build")
$ npm run deploy
```
